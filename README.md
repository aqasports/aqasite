- Click **🔄 RECHARGER** next to each file

### Batch Operations
- Edit multiple coaches at once
- Click **GÉNÉRER** one time
- Click **DÉPLOYER TOUT** to get both files

## 🆘 Troubleshooting

**Q: Files not auto-loading?**
A: Check browser console (F12). Files must be in same folder. Use manual paste as fallback.

**Q: GÉNÉRER not working?**
A: Ensure files are loaded. Check coach names don't have special characters.

**Q: Downloads not appearing?**
A: Check browser downloads folder. Try 📋 COPIER first to verify content.

**Q: Not sure what changed?**
A: Watch for ⚠ **MODIFIÉ** indicators next to filenames.
